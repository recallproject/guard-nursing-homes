Changes from current (facility rows):
① Hook lines moved ABOVE tags — the most compelling data point is now the first thing you read after the name, not buried below
② "deficiencies" → "violations" — plain English, no healthcare jargon

Summary of All Recommended Changes

P1 — Do First

"Avg Score" → "Avg Risk Score"
Add gradient bar + "higher = more risk" context. Families currently have zero context for what 30.0 means.
"deficiencies" → "violations" site-wide
Healthcare jargon that means nothing to families. "Violations" is universally understood.

P2 — High Impact

Hook lines above tags, not below
The most compelling data ("9.8% of days had zero RNs") should be the first thing you read. Currently it's last.
Add breadcrumb navigation
Home › Map › South Dakota. Orients the user and gives escape paths.

P3 — Polish

Plain-language sort labels
"Risk Score (High to Low)" → "Most Concerning First"
Tag visual hierarchy
Harm/danger tags get colored borders so they pop. Neutral counts recede.
Star rating aria-labels
Screen reader support: "1 out of 5 stars" instead of meaningless symbols.

✓ What's Already Working

Red score badges — instantly scannable risk at a glance
Hook lines — the most powerful feature on this page. "9.8% of days with zero RNs" changes decisions.
List/Grid toggle — power users love options
Trend arrows — subtle but meaningful for repeat visitors
