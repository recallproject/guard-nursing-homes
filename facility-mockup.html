Glossary Updated
Terms used throughout this report
NEW — Searchable + Expanded
TermMeaning
RNRegistered Nurse — highest-level bedside nurse, responsible for assessments and care decisions
LPNLicensed Practical Nurse — administers medications, wound care, vital signs
CNACertified Nursing Assistant — provides daily personal care (bathing, feeding, dressing)
Hrs/res/dayTotal nursing hours divided by number of residents, per day — the key staffing density metric
PBJPayroll-Based Journal — mandatory payroll records submitted to CMS (not self-reported)
SFFSpecial Focus Facility — CMS designation for the worst-performing ~1% of nursing homes
CCNCMS Certification Number — unique federal ID for each Medicare-certified facility
F-tagFederal regulation code (e.g., F689 = accident hazards). Each deficiency cites a specific F-tag.
K-tagFire safety regulation code — K-level citations are the most serious fire code violations
REITReal Estate Investment Trust — a corporate structure where the building is owned separately from operations
Scope & SeverityCMS grid classifying each deficiency by how many residents affected and how serious the harm
Adds search/filter functionality and 3 new terms (F-tag, K-tag, REIT, Scope & Severity) that appear elsewhere on the page. Current glossary has 9 terms — expanded to 11.
